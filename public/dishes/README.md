# Photos de plats Le Carré

Placez ici 3 photos de plats du restaurant Le Carré :

1. `plat1.jpg` ou `plat1.png` - Premier plat
2. `plat2.jpg` ou `plat2.png` - Deuxième plat  
3. `plat3.jpg` ou `plat3.png` - Troisième plat

## Formats acceptés
- JPG / JPEG
- PNG
- WebP

## Recommandations
- Résolution : 1080x1080 ou plus (carré)
- Qualité : Haute résolution pour un meilleur rendu
- Nom des fichiers : `plat1.jpg`, `plat2.jpg`, `plat3.jpg`

Une fois les photos ajoutées, elles seront intégrées dans la vidéo 3D.
